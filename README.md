# 🚀 Accredian Enterprise - Landing Page Clone

A partial clone of the Accredian Enterprise website built using **Next.js (App Router)**, **TypeScript**, and **Tailwind CSS**.
This project focuses on recreating a modern landing page experience with clean UI, reusable components, and responsive design.

## ⚙️ Setup Instructions

```bash
# Clone the repository
git clone https://github.com/Aditya-dott/accredian.git

# Navigate into the project
cd accredian

# Install dependencies
npm install

# Run development server
npm run dev
```

Visit: http://localhost:3000

## 🧠 Approach

* Broke down the UI into reusable components such as Navbar, Hero, Features, Testimonials, and Footer
* Used Next.js App Router for scalable and clean routing
* Implemented smooth scrolling using anchor-based navigation
* Focused on responsiveness, clarity, and modular design
* Added an About page to demonstrate routing capability
